# STEEL SENTINEL

## O projekcie

System Steel Sentinel to system monitoringu infrastruktury krytycznej dla wojska oraz służb ratowniczych. W dobie zagrożeń hybrydowych i wojny dronowej, monitorowanie kluczowych obiektów jest niezwykle ważne. Główną cecha systemu jest zasada offline-first. Oznacza to, że system jest zaprojektowany tak, aby działać bez dostępu do internetu.
